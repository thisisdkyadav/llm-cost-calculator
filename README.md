# LLM Cost Calculator

A premium, modern, and responsive web application to calculate API costs for Large Language Models (LLMs) with high precision.

## ‚ú® Features

- **Real-time Calculations**: Results update instantly as you adjust pricing or token counts.
- **High Precision**: Handles micro-costs (e.g., $0.000015) with scientific accuracy up to 6 decimal places.
- **Requests per Dollar**: Automatically calculates how many identical requests you can process for $1 USD.
- **Premium Design**: Sleek dark-mode interface with glassmorphism effects and animated gradients.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop screens.

## üìñ Usage

1.  **Input Pricing**: Enter the cost per 1 million tokens for both Input and Output (provided by your LLM provider like OpenAI or Anthropic).
2.  **Input Usage**: Enter the number of prompt (input) and completion (output) tokens.
3.  **View Results**: The total cost, cost breakdown, and requests-per-dollar metric will update automatically.
